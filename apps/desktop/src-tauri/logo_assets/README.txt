neuink Logo Asset Pack

Contents:
- neuink_app_icon.svg: Windows/macOS app-icon source with a full-size blue rounded-square background.
- neuink_logo_vector.svg: editable vector SVG with transparent background.
- neuink_logo_vector_whitebg.svg: SVG with white background.
- neuink_icon_16x16.png ... neuink_icon_1024x1024.png: transparent PNG icons.
- neuink_icon_whitebg_16x16.png ... neuink_icon_whitebg_1024x1024.png: white-background PNG icons.
- neuink_favicon_multi_size.ico: multi-size favicon/icon file.
- asset_manifest.json: asset list and brand blue value.

Recommended use:
- Desktop app / taskbar icon: neuink_app_icon.svg, rendered through `npm run tauri -- icon`.
- App icon / product icon: 512x512 or 1024x1024 PNG.
- Website favicon: neuink_favicon_multi_size.ico or 32x32 PNG.
- Design editing / further refinement: neuink_logo_vector.svg.
